## CUDA 입문

수강 전 주의 사항
1. 윈도우11, VS2022, 엔비디아 RTX GPU 기준으로 제작되었습니다. 이론적으로는 리눅스에서도 공부를 하실 수 있으나 빌드 설정 등을 스스로 해결하셔야 하기 때문에 매우 번거롭습니다.
2. 기본적인 프로그래밍 연습이 어느정도 되어 있는 분들을 대상으로 간결하게 제작되었습니다. 알고리듬이나 그래픽스 수강 후에 시작하시기를 권장합니다. 초보자에게는 권하지 않습니다.
